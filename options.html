AJ's Philo Guide + Kaslana's Auto-Type + Philo-NoDrag

Some tool took our extension's features that were free and open, then sold them without giving credit. So we remade it from scratch, cleaned it up, and gave it back to the community. For free this time.

The Story

Original Goal

AJ's Philo Guide brought a more efficient word search, Philo NoDrag brought Type‑to‑Fill. Kaslana handled Auto‑Type. For free, for the students.

Then some tool came along, took our ideas, obfuscated the code, added license checks, and made it paid. Not cool.

So here we are. We removed the paywalls, fixed the UI/UX, and released it again. Free and open..

What It Does

Smart matching for single/multiple choice questions
Save and load answer presets for reuse
JSON import/export for data sharing
Docked panel interface. Doesn't get in your way
Inline typing interface for direct input

How to Use

1
Go to quiz or review page. It detects it automatically
2
The panel should show up.
3
Paste answers or use presets. Use "||" to separate multiple answers
4
Click "Apply". It handles the typing
5
Save/Export presets for next time
